# tsc-tips
Tips when using typescript

1.8 and 2.1 compatible tips in seperate folders

### Setup

    cd 1.8
    npm install
    cd 2.1
    npm install
    
### Use

Uncomment code where instructed and compile to see the errors that are caught

    cd 1.8
    npm run compile 2> /dev/null
    
 or
 
    cd 2.1
    npm run compile 2> /dev/null
